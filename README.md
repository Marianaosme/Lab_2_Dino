# Dino Game 
**Laboratorio 2**  
Sistemas Embebidos · Universidad EIA ·  

## Uso de los colores
| Elemento   | Color |
|------------|--------|
| Dinosaurio | Verde |
| Obstaculo  | Rojo |
|Game over  | Pantalla Roja |

## Cómo se juga
- Evitar chocar con los obstáculos, para esto utiliza el botón para controlar el salto del Dino (led Verde)
- Si el dino no salta a tiempo y colisiona con el obstáculo, el juego se reinicia automáticamente.
